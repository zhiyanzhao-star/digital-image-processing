# 图像变形与融合 (Image Warping and Blending)

本仓库实现了数字图像处理中的三种经典图像变换技术：基于控制点的图像变形（MLS）、仿射变换以及泊松图像融合。

## 环境要求

```setup
pip install gradio opencv-python numpy pillow torch torchvision
```

- Python 3.8+
- OpenCV (`cv2`)：图像读写与几何变换
- NumPy：数值计算
- Gradio：交互式 Web 界面
- PyTorch：泊松融合中的梯度优化

## 程序说明

### 1. 基于控制点的图像变形 — `point_transform.py`

基于 **Moving Least Squares (MLS)** 算法的点引导图像变形。

**原理：**
- 用户在图像上交替点击源控制点和目标控制点（至少 3 对）
- 对目标图像中的每个像素，根据其到各目标控制点的加权距离，计算加权中心与最优旋转矩阵
- 通过 SVD 分解得到局部最优刚性变换，反向映射回源图像进行采样
- 蓝色点标记源位置，红色点标记目标位置，绿色箭头指示对应关系

**运行方式：**
```bash
python point_transform.py
```

### 2. 图像仿射变换 — `globe_transform.py`

支持缩放、旋转、平移和水平翻转的交互式图像变换工具。

**原理：**
- 将缩放、旋转、翻转、平移分别构建为 3×3 仿射变换矩阵
- 按正确顺序组合：平移 @ 翻转 @ 旋转 @ 缩放（从右到左应用）
- 使用 OpenCV 的 `warpAffine` 完成变换，白边填充

**运行方式：**
```bash
python globe_transform.py
```

### 3. 泊松图像融合 — `run_blending_gradio.py`

基于泊松方程的无缝图像融合。

**原理：**
- 用户在前景图像上绘制多边形选区，指定在背景图像上的放置位置
- 使用拉普拉斯算子提取前景区域的梯度场
- 以背景图像边界为狄利克雷边界条件，通过 Adam 优化器最小化拉普拉斯损失
- 迭代 5000 步，在后 1/3 阶段将学习率降为原来的 0.1 倍
- 损失函数：前景区域与融合区域拉普拉斯响应的 L1 损失

**运行方式：**
```bash
python run_blending_gradio.py
```

## 结果展示

### 控制点变形效果
- 演示视频：`point_transform.mp4`
- 通过 MLS 算法实现平滑的图像局部变形，保持局部刚性

### 仿射变换效果
- 演示视频：`globe_transform.mp4`
- 实时响应用户调整的缩放、旋转、平移与翻转参数

### 泊松融合效果
- 将选定的前景区域无缝融合到背景图像中，保持前景纹理的同时平滑过渡边界

## 实验小结

| 方法                   | 核心算法                     | 交互方式            | 适用场景               |
| ---------------------- | ---------------------------- | ------------------- | ---------------------- |
| 点引导变形 (MLS)       | Moving Least Squares + SVD   | 点击控制点对        | 图像局部变形、人脸编辑 |
| 仿射变换               | 矩阵乘法组合 3×3 仿射矩阵    | 滑块调节参数        | 图像几何校正、缩放旋转 |
| 泊松融合               | 拉普拉斯梯度场 + Adam 优化   | 多边形选区 + 滑块   | 无缝拼接、图像合成     |

三种方法均基于 Gradio 构建交互式界面，可在浏览器中直观地体验数字图像处理中图像变形与融合的核心概念。
